# Forge Command

**Official App Name for iOS and iPadOS**

## App Identity

- **Name:** Forge Command
- **Platforms:** iOS 16.0+, iPadOS 16.0+, macOS 13.0+
- **Category:** Developer Tools / Team Coordination
- **Type:** Fleet management and operational coordination

## Description

Forge Command is a real-time coordination platform for distributed engineering teams. Monitor your fleet of agents, manage communication channels, track operational streams, and maintain situational awareness across your entire infrastructure.

### Key Features

- **Fleet Management** - Real-time monitoring of agents organized by department with health status, watchlist tracking, and exception alerts
- **Inbox** - Centralized coordination stream with mentions, broadcasts, and priority messages
- **Channels** - Team communication organized by topics with threaded conversations
- **Director Console** - Command center for quick actions and operational control
- **Blackboard** - Shared task and item management

### Technical Stack

- **Backend:** forge-graphd (Rust coordination daemon)
- **Transport:** HTTP + Server-Sent Events over Tailscale
- **API:** RESTful endpoints at :9090
- **Architecture:** SwiftUI with Swift Concurrency (async/await)
